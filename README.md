# Aurore Portfolio

A Vite + React portfolio with scroll-snapped sections.

## Tech stack

| Layer | Technology |
| --- | --- |
| App framework | React |
| Build tool | Vite |
| Styling | CSS |

## Project structure

```text
my-portfolio/
|-- index.html
|-- package.json
|-- vite.config.js
|-- css/
|   |-- reset.css
|   `-- style.css
`-- src/
    |-- main.jsx
    `-- portfolioContent.js
```

## Run locally

```bash
npm install
npm run dev
```

Then open the localhost URL Vite prints in the terminal.

## Build

```bash
npm run build
```

The production files are written to `dist/`.

## Customize

Edit `src/portfolioContent.js` for portfolio text and profile cards.

Edit `css/style.css` for typography, layout, and UI styling.
